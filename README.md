### README

## アプリ名
タイムスケジュールを利用したメール送信アプリ

## 概要
laravelを使用して、登録したメール文を送信するアプリを作成。

## 使用技術
- html
- php バージョン：PHP 7.3.24
- mysql
- laravel バージョン：Laravel Framework 6.20.4
- apache　バージョン：Apache/2.4.6
- centos バージョン：CentOS Linux release 7.7.1908 (Core)
- javascript
- virtualBox

## 実装した機能
- 基本的なCRUD
- Validation
- メール
- 認証モデルの変更
